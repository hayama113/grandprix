# Grand Prix Quiz v1.7

コース選択対応版です。

## コース

1. F1・クルマ総合
   - 既存10,000問

2. F1レース別出来事・マニア
   - 1990年〜現在の主要レース出来事
   - タイトル決定戦、雨、赤旗、チームオーダー、ペナルティ、メカニカル深掘り
   - 出典メタデータ付きスターター問題

3. 生活の知恵・雑学トリビア
   - 生活トリビア500問を別コースとして統合

## 注意

v1.7は「構造とスターター問題集」です。
1990年〜現在までの全レースを記事単位で完全網羅するには、追加バッチでレースごとの出典確認と問題追加が必要です。

## 必須ファイル

- index.html
- style.css
- app.js
- questions.json
- question-data.js
- manifest.json
- service-worker.js
